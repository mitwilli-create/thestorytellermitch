Flows Template Pack A · Multi-Market Ad Variant

Built as part of the One Script, Seven Markets spec bundle for the ElevenLabs Forward Deployed Creative role. Cost figures are measured actuals from the bundle's logged receipts (2026-07-13) or published list rates as cited inline. MERIDIEM is a fictional canned Spanish white wine created for the demo. No company named in this document is claimed as an ElevenLabs client.

Template description (two sentences): Takes one master ad script plus transcreated copy for each target market and outputs a finished per-market voice track and a market-adapted music bed, all in the same brand voice. Built for brand and agency teams running one campaign across DE, FR, and ES who currently re-book talent and re-license music for every market.

Companion proof: the MERIDIEM demo set in this bundle (a Spanish master plus localized exports, EN, DE, FR, IT, PT, and NL among them) was produced with exactly this workflow. Measured incremental cost per added market was under fifteen cents for voice, music, and ambience; receipts in the bundle's receipts/ folder.

1. What the template does

  • In: master script (text), one transcreated script per market (text), one brand voice, one master visual edit (video upload), one music direction per market (text prompt).
  • Out: per-market composition (visuals + localized voice + market music), bundled as a zip when multiple outputs are selected.
  • Invariant: the voice is the SAME voice in every market. Voice consistency is the brand asset; the language changes, the voice does not.
  • Deliberate choice: market copy is transcreated by a human (or an LLM step outside the flow), never machine-translated word for word. The template treats localized copy as an input, because idiom is a creative decision. See the localization notes in the bundle's localized-scripts/ for worked examples ("con la chispa justa" becomes "mit genau der richtigen Perlage" because Perlage is how German wine copy actually talks about sparkle; the candid-photos beat lands in Dutch as "foto's die vanzelf ontstaan"; and so on).

2. Prerequisites (5 minutes)

  1. ElevenLabs account with ElevenCreative access and enough credits (a five-market run of a 25 second spot uses roughly 2,000 to 4,000 credits total: TTS at 1 credit per character, music at 900 credits per minute).
  2. Your brand voice available in the workspace (an Instant or Professional Voice Clone, or a licensed library voice).
  3. The master visual edit exported as a single mp4 (visuals only or with diegetic sound effects mixed in, no voice, no music).
  4. Transcreated copy per market, one short paragraph or beat list each.

3. Click-by-click build (target: under 25 minutes)

Create the canvas

  1. Open ElevenLabs, go to the Creative section, open Flows, and create a new flow. Name it Multi-market ad variant.

Master lane (build once)

  1. Right-click the canvas (or use the bottom toolbar) and add a Text node. Paste the master EN script. Rename the node Script EN.
  2. Add a Text to Speech node. Drag from the output port of Script EN to the text input port of the TTS node. In the TTS node settings, select your brand voice and an Eleven v3 family model (v3 handles multilingual delivery well). Rename it VO EN.
  3. Add a Music node. In its prompt, describe the master bed (for MERIDIEM: warm organic downtempo, live-feel instruments, golden hour mood, instrumental bed under narration, no melody hook). Set length to your spot length plus 2 seconds. Rename it Music EN.
  4. Add an Upload Media node and upload the master visual edit mp4. Rename it Master visuals.
  5. Add a Composition node. Wire Master visuals, VO EN, and Music EN into it. In the composition, keep the voice on top: music sits well under the voice at all times. Rename it Spot EN.
  6. Hover the Run control on each node to see its credit estimate, then run the four nodes in order (Text nodes are free; TTS, Music, and Composition each show their cost before you commit). Verify: you can play Spot EN inside the canvas and the voice sits clearly on top.

Market lanes (duplicate and swap, about 3 minutes per market)

  1. For each market (DE, FR, ES): add a Text node with that market's transcreated script, a Text to Speech node wired from it (SAME brand voice, same model), and a Music node with the market-adapted prompt. For the demo set the shifts were: DE minimal warm deep house, precise and unhurried; FR jazz-tinged, brushed drums, aperitif mood; PT guitarra-tinged, warm and easy. (In the demo the Spanish cut IS the master, so the master market simply keeps the master bed.) Keep tempo family and energy curve close to the master so the same edit still lands.
  2. Add a Composition node per market. Wire in the SAME Master visuals node (one upload feeds every market lane), plus that market's VO and Music nodes.
  3. Run only the new nodes. Rerunning a single node never touches the rest of the canvas, so iterating on one market's music costs only that node's credits.

Publish as a template

  1. Open the flow's template controls. Select as inputs: every market Text node, the voice selection on the TTS nodes, and the Master visuals upload node. Select as outputs: every market Composition node (multiple outputs download bundled as a zip).
  2. Run the whole template once end to end (publishing requires at least one successful run).
  3. Save, then set sharing: Workspace for the client team, or Link only to hand it to an account executive for a demo.

4. Handoff notes for the team that receives this

  • To add a market, duplicate one market lane and change two things: the script text and the music prompt. Nothing else moves. Our measured marginal cost for the IT market was about 13 cents of API-equivalent spend for voice, music, and ambience combined.
  • If the localized voice read sounds flat in one language, fix the copy first (read it aloud), then regenerate only that TTS node.
  • Keep a one-line change log in the flow description so the next operator knows which market prompts were tuned and why.

5. Known seams (honest limits)

  • Flows composes and layers; it is not a timeline editor. If a market's VO runs long enough to need a re-cut of the visuals, export to ElevenCreative Studio for timeline work, then bring the revised master mp4 back into the Master visuals node.
  • Word-timed captions are a Studio step (see Template Pack B).
  • UI labels evolve. If a node name here does not match what you see, the shape of the build does not change: text feeds speech, prompts feed music, one shared visuals upload feeds every market composition. The ElevenCreative Flows docs page is the source of truth.
