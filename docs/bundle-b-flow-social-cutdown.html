Flows Template Pack B · Social Cutdown

Built as part of the One Script, Five Markets spec bundle for the ElevenLabs Forward Deployed Creative role. Cost figures are measured actuals from the bundle's logged receipts (2026-07-13) or published list rates as cited inline. MERIDIEM is a fictional canned Spanish white wine created for the demo. No company named in this document is claimed as an ElevenLabs client.

Template description (two sentences): Takes one finished hero spot and turns it into three vertical social cutdowns (a 6 second hook, a 10 second product cut, and a 15 second story cut) with market-correct captions and a re-voiced hook line per cutdown. Built for social and performance teams who get a hero ad from the brand team and need platform-ready variants the same day.

1. What the template does

  • In: hero spot (video upload), three hook lines (text, one per cutdown), brand voice, target market caption language.
  • Out: three vertical cutdowns, captioned, each with its own re-voiced hook over the opening shot.
  • Why re-voice the hook: a cutdown that just truncates the hero loses its opening logic. Writing a fresh 6 to 10 word hook per cutdown and generating it in the same brand voice costs a few credits and makes each cutdown open like it was built for the placement.

2. Where Flows ends and Studio begins (read this first)

Flows is a generation canvas; trimming and word-timed captions are timeline work, which lives in ElevenCreative Studio. This template therefore has two halves, and that is by design, not a workaround:

  • Flows half: generate the per-cutdown hook VO and any end-card assets. Publish this as the reusable template.
  • Studio half: trim the hero to the three cutdown lengths, drop in the hook VO, and generate captions. Documented below as a fixed 10 minute checklist the operator repeats per campaign.

3. Click-by-click: the Flows half (target: under 15 minutes)

  1. Create a new flow named Social cutdown kit.
  2. Add an Upload Media node and upload the hero spot mp4. Rename it Hero spot. (It rides along as a template input so the operator always has the source in the same canvas.)
  3. Add three Text nodes: Hook 6s, Hook 10s, Hook 15s. Paste one hook line into each. For the MERIDIEM demo: Golden hour is not going to wait. (6s), A Spanish white, cold from the can. (10s), The day does not end. It ripens. (15s).
  4. Add three Text to Speech nodes, one wired from each hook Text node, all set to the SAME brand voice used in the hero. For localized cutdowns, paste the transcreated hook in the market language into the Text node; same voice, market language.
  5. Optional: add an Image Generation node for a platform end card (a 9:16 product still with the tagline; use a text-capable image model so the tagline stays legible) and a Sound Effects node for one soft closing accent. Keep any accent low, rounded, and under the voice.
  6. Run each node once (hover Run to see the credit estimate; TTS on a hook line costs under 60 credits).
  7. Publish as a template: inputs are the three hook Text nodes, the voice selection, and the Hero spot upload; outputs are the three TTS nodes plus the optional end-card image (multiple outputs bundle as a zip). Run once end to end, save, share to Workspace.

4. The Studio half: fixed checklist (about 10 minutes per campaign)

  1. Export or send the hero spot into ElevenCreative Studio (from the flow, use the export to Studio option so assets land in the same project).
  2. Duplicate the hero timeline three times, named 6s, 10s, 15s.
  3. Trim each: the 6s cut keeps the strongest visual beat plus the end card, the 10s cut keeps product beats plus the close, the 15s cut keeps the story spine (open, product reveal, close). Front-load the shortest shots; nothing holds past 4 seconds.
  4. Drop the matching hook VO (from the Flows zip) over each cut's opening shot, replacing the hero VO there. Duck music under it.
  5. Generate captions per cutdown in the market language, check every line against the audio (numerals, brand name casing, no clipped words), and burn in or export as a sidecar per platform spec.
  6. Export all three as 1080x1920 mp4. Verify each opens with sound OFF and still communicates (captions carry the hook).

5. Handoff notes

  • The template is language-agnostic: a DE cutdown set is the same flow with DE hook lines pasted in and DE captions in Studio. Voice stays the brand voice.
  • Keep hooks under 10 words. If the TTS read crowds the opening shot, shorten the words, not the audio.
  • Repeat-use pattern: the social team keeps one saved flow per brand (voice locked), and per campaign only the hero upload and three hook lines change.

6. Known seams (honest limits)

  • Precise caption timing and any frame-level trim will always be Studio work; the template's value is that everything generated (voice, end card, accent) is one reusable, credit-transparent canvas away.
  • If Studio's caption language list is missing a target market, generate the SRT externally and import it; do not ship machine captions unchecked in a language nobody on the team reads. That rule is in the workshop kit's objection FAQ for a reason.
  • UI labels evolve; the docs page for ElevenCreative Flows and Studio is the source of truth.
