The Same Workflow, No Compliance Tier

A worked plan, not a produced video: it maps the measured MERIDIEM workflow onto a non-alcohol category so the generalization is inspectable. Cost lines marked "measured" are actuals from the bundle's logged receipts; lines marked "projected" apply those same measured rates to this plan. TARDEO is a fictional Spanish snack brand created for this page. No company named here is claimed as an ElevenLabs client.

The MERIDIEM demo is deliberately set in the hardest consumer category: alcohol, where France's Loi Evin and Italy's sport-association rules force a three-tier compliance structure. A fair question follows: how much of the workflow is really about the category? This page answers it by working the same brief for TARDEO, a fictional Spanish toasted-corn snack, a category with no alcohol rules at all. Snacking, beauty, fashion, mobility: the mapping below holds for any category where the product is identical across markets but the moment of use is cultural.

What carries over unchanged

  • The master-plus-exports structure. One home-market master (a Madrid plaza at the tardeo hour, the late-afternoon social ritual the brand is named for), then per-market exports that swap the voice track, the music bed, and the culturally-marked wardrobe in the two people beats. The film itself never changes.
  • The cloned brand voice across every market. Same voice_id, per-beat TTS calls, localized copy per market. Nothing in that chain knows or cares what the product is.
  • The QA gates. Frame-level label pass, blind audio review, persona board, caption verification. The label gate matters in every category: for MERIDIEM it caught a hallucinated alcohol declaration; for a snack it would catch invented nutrition claims, which EU food-information rules (Regulation 1169/2011) police just as hard.
  • The templates. Both Flows packs run as-is: the multi-market ad variant and the social cutdown are category-blind. So is the API version.

What drops

  • The entire compliance tier. No sport-free re-dress for France and Italy, no product-only French cut, no per-market counsel pass on lifestyle framing. Seven markets ship the same single tier.
  • The category disclaimers. No responsible-consumption line, no health warning block reserved in the end card.

The honest difference from the alcohol version is not the generation bill, it is the removed coordination: no counsel review per market, no second wardrobe pass for two markets, no separate French cut to QA. In the MERIDIEM set those compliance variants cost 0 USD of generation but real review time; here that line disappears entirely. The category-specific work that remains is the same work every category has: transcreation of eight lines per market and a label-accuracy gate tuned to the category's regulator.

Run it yourself: the multi-market template takes this plan as its inputs without modification (swap the scripts and the master visual; the voice, music, and composition lanes do not change). The receipts pattern that prices it to the cent is in the API appendix.
