voice-osSHIPPED

Voice fidelity you can measure

Most LLM voice matching is vibes-based prompting that cannot say how close a draft is to the target. This Python pipeline scores drafts against a voice corpus on six stylistic axes, with dual-persona routing and a QA gate that blocks output below threshold. Fidelity is quantified, not asserted.

6.9M+ word calibration corpus · pytest suite · deterministic offline mode

Triage that knows when to stop

Internal comms intake from senior engineers is high volume and judgment-heavy. This agent triages, revises, and escalates on Google Apps Script and Gemini: three sequential prompts, conditional KB loading, and a confidence threshold that holds low-confidence items for human review. Scoped prompts with defined escape hatches, not one mega-prompt. Published as a sanitized public demo.

Three-prompt architecture · dry-run harness · per-submission decision logging

monolithSHIPPED

Prompt-engineered guardrails can be talked out of their own rules. This deterministic wardrobe architect and budget controller is a TypeScript PWA where the LLM only does perception; three code-level constraint gates issue the verdict. Three-gate verdict pipeline · 50 pure-function tests · runs without an API key
